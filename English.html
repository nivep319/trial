Tenses:

Rules of Tenses help one understand how to correctly use the different tenses in a sentence, without making a grammatical mistake and also by easily indicating when an event or action has occurred.

There are three types of tenses:

  • Present tense
  • Past Tense
  • Future tense

All the three types of courses are furthur subdevided into:

  • Continuous tense
  • Perfect tense
  • Participle tense

Formulae to remembar:

  • 1.Simple Present:(+) S + V1 + O/C
  • 2.Simple Past: S + to be (was/were) + O/C S + V2 + O/C
  • 3.Simple Future: S + will/shall + V1 + O/C S + am/is/are + going to + V1 + O/C
  • 4.Present Comtinuous: S + be + V1 + ing + O/C
  • 6.Future Continuous: (+) S + will/shall + be + V-ing + O/C
  • 7. Present Perfect : (+) S + has/have + been + O/C
  • 8.Past Perfect: (+) S + had + V3 + O/C
  • 9.Future Perfect : (+) S + shall/will + have + been + C
  • 10.Present Perfect Continuous Tense (+) S + have/has + been + V-ing
  • 11. Past Perfect Continuous: (+) S + had + been + V-ing + O/C.
  • 12.Future Perfect Continuous: S + shall/will + have + been + V-ing + O
